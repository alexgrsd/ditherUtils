# Mon pitit logiciel d'image processing :D

# Kessecé ???

Pour l'instant est en WIP, a comme objectif de créer a partir d'images et de vidéos divers effets comme de l'ascii art exporté en image ou en vidéo, ainsi que divers effets de post processing comme du glitch (RGB shift, etc....)

# Comment me procurer ce délicieux programme ???

Il y a simplement a le clone depuis la main et a le compiler avec le magnifique fichier cmake présent ! 
Cependant, je ne garanti aucune compaptibilité avec d'autres machines que la mienne, je développe sur macOS 26.1 avec OpenCV (nécessaire pour la compilation attention)

Pour l'instant pas grand choses mais des futures améliorations arriveront peut etre :
- Support pour une imput en vidéo
- De la couleur
- Des effets de glitch
- Plus d'effets de base autre que l'ascii art qui pourraient être sympas

Pour l'instant le logiciel ne fait que du threshold, pour avoir de bons résultats il est recommandé de soit les threshold en avance soit d'utiliser des logiciels de dithering (dither boy, etc) pour mieux controler l'input, des algo de dither seront peut être ajoutés dans le futur cela dit !!!
